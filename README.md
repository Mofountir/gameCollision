# 🎮 Jeu de Collision

Bienvenue dans le Jeu de Collision ! Guidez le petit monstre jusqu'à la sortie tout en évitant les obstacles.

## 🚀 Installation

1. Clonez ce dépôt :
    ```sh
    git clone <URL_DU_DEPOT>
    ```
2. Ouvrez le fichier `src/index.html` dans votre navigateur pour jouer.

## 📂 Structure du Projet

- `src/index.html` : Fichier HTML principal.
- `src/style.css` : Styles CSS.
- `src/js/` : Scripts JavaScript.

## 📜 Règles du Jeu

- 🕹️ **Déplacement** : Utilisez les touches fléchées pour déplacer le joueur.
- 🚫 **Obstacles** : Évitez les obstacles rouges.
- 🏁 **Objectif** : Atteignez la sortie dorée pour gagner.
- 🔄 **Retour au Départ** : Si vous touchez un obstacle rouge, vous revenez au point de départ.

Amusez-vous bien ! 🎉
